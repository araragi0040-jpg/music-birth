Humming Pitch Viewer

ドレミバー

鼻歌や歌声の音程を、カラオケの音程バーのように見える化します。 v018では、音声ファイルを選択できない問題を修正しました。Basic Pitch高精度解析と従来解析の両方を利用できます。

1

音源を用意する

ファイルから読み込む

すでに録音済みの音声ファイルを使います。

音声ファイルを選択 MP3 / WAV / M4A / AAC / OGG / WebM など

その場で録音する

鼻歌・歌声を録音して、そのまま解析できます。

待機中 0:00

録音機能はブラウザのマイク許可が必要です。 公開URLやlocalhostでは動きやすく、環境によっては直接HTMLを開くと録音できない場合があります。

おすすめ設定
高精度解析は、SpotifyのBasic Pitchモデルを端末内で動かします。 初回はモデル読み込みがあり、通常解析より時間がかかります。音源が外部サーバーへ送信されることはありません。

基本は「高精度（Basic Pitch）」＋「おまかせ」＋「やさしい表示」＋「横スクロール」でOKです。全体表示は、曲全体の流れをざっくり見たい時に使えます。

必要なときだけ詳細調整
うまく出ない時だけ細かく調整

音が出ない → 「声が小さいとき」または「音の拾いやすさ：高め」

バーが細切れ → 「バーをなめらかにしたいとき」または「解析の細かさ：ざっくり」

雑音を拾う → 「雑音が多いとき」または「音の拾いやすさ：低め」

音源を選択するか、録音してください。

録音音源の書き出し

録音した声そのものを保存できます。作曲アプリに入れる場合は、互換性の高いWAV保存がおすすめです。

録音または音源読み込み後に保存できます。保存ボタンは常に押せますが、データがない場合は理由を表示します。
2

音程バーを見る

左のドレミが音の高さ、右方向が時間です。

タップで選択、上下ドラッグで高さだけ修正できます。

スマホでは、音程バー枠の中だけ横スクロールします。

解析結果はここに表示されます。
高さを直す： オレンジ色のバーを押したまま上または下へ動かします。 修正内容は再生・画像保存・ドレミ音WAV保存に反映されます。

キーボード風は、ピアノほどリアルではありませんが、電子音よりも柔らかく減衰する音色です。

ドレミ音の書き出し

解析された音程バーを、現在選択しているドレミ音色でWAV保存できます。

解析後に保存できます。まだ解析結果がない場合は理由を表示します。

音の流れ

スマホで見にくい時は、ここでドレミの並びを確認できます。

解析後に、音の流れがここに表示されます。
バーをクリックすると、音名と時間が表示されます。

ポイント

  • 単音の鼻歌・歌声・メロディ確認向けです。
  • まずは「高精度（Basic Pitch）」＋「おまかせ」で試すのがおすすめです。
  • 細かいズレは、必要な部分だけバーを上下ドラッグして直せます。
  • 作曲アプリへ渡すなら「録音音源をWAV保存」または「ドレミ音をWAV保存」が便利です。
  • 伴奏入り音源や和音は、正確に読めないことがあります。
