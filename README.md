# ドレミバー v011

鼻歌・歌声・単音メロディの音程を、カラオケの音程バーのように表示する簡易Webアプリです。

v011では、音声ファイルの読み込み・ブラウザ録音・解析モード・音名表示切替・画像保存・録音音源WAV保存に加えて、ドレミ音をキーボード風に改善し、解析されたドレミ音もWAV保存できるようにしました。

## 使い方

### ファイルから使う場合

1. `index.html` をブラウザで開く
2. 音声ファイルを選択
3. 「解析する」を押す
4. 音程バーが表示される

通常は「解析モード：おまかせ（おすすめ）」＋「音名表示：やさしい（ドレミだけ）」のままで大丈夫です。

### 録音して使う場合

1. 「録音開始」を押す
2. マイク使用を許可する
3. 鼻歌・歌声を録音する
4. 「録音停止」を押す
5. 録音音源を再生確認する
6. 「解析する」を押す

## 録音機能について

録音機能はブラウザのマイク機能を使います。

環境によっては、`index.html` を直接開いた場合に録音できないことがあります。
その場合は、以下のどちらかで試してください。

### 方法1：Vercelなどに公開する

HTTPSの公開URL上ではマイク機能が動きやすいです。

### 方法2：ローカルサーバーで開く

このフォルダで以下を実行します。

```bash
python -m http.server 8000
```

その後、ブラウザで以下を開きます。

```text
http://localhost:8000
```

- E = ミ
- F = ファ
- G = ソ
- A = ラ
- B = シ

## v008 の変更点

### スマホでの見やすさ改善

- スマホ表示時は音程バーの横幅を少しコンパクトに調整
- スマホ表示時は行間を少し広げて、バーとドレミの高さを見やすく調整
- 横スクロールしても音名がわかりやすいように、音名をバー内の途中にも薄く表示
- 音程バーの下に「音の流れ」を追加
  - ド / レ / ミ の並びをチップで確認できます
  - チップを押すと、その位置から元音源を再生できます

### ドレミ音の音量改善

- ドレミ音で再生する時の音量を上げました
- 音割れしにくい範囲で、以前より聞こえやすい設定にしています

## スマホ確認のポイント

- 解析後、横スクロールしても音名が見えるか
- 「音の流れ」でドレミの並びが確認しやすいか
- チップを押して該当箇所から再生できるか
- ドレミ音で再生した時、以前より音量が十分か


## v009 の変更点

### 音程バー表示の方針変更

v008ではスマホ向けに音程バーを少し圧縮していましたが、v009では基本を横スクロール表示に戻しました。

- 基本：横スクロール（おすすめ）
- 補助：全体表示

全体表示は、全体の流れをざっくり見たい時に使います。
細かく音程を見る場合は、横スクロール表示がおすすめです。

### ドレミ音量の改善

ドレミ音が小さく聞こえる問題に対して、以下を変更しました。

- 音量設定を追加
  - 標準
  - 大きめ
  - 最大（スマホ向け）
- 初期値を「最大（スマホ向け）」に設定
- 正弦波だけではなく、三角波＋小さな倍音を追加
- スマホスピーカーでも聞こえやすい明るい音色に変更
- 音割れを抑えるためにコンプレッサーを追加

音が大きすぎる場合は、画面上の「ドレミ音量」を下げてください。

## v009確認ポイント

- 音程バーが基本横スクロールで見やすいか
- 「全体表示」に切り替えた時に全体感が確認できるか
- ドレミ音量「最大（スマホ向け）」で十分聞こえるか
- 音が大きすぎる場合に「大きめ」「標準」で調整できるか


## v010 の変更点：録音音源の書き出し

録音した音声データを保存できるようにしました。

### 追加ボタン

- 元の音源を保存
- WAVで保存

### 元の音源を保存

ブラウザが録音した形式のまま保存します。

例：

- Chrome / Android：webm になることが多い
- Safari / iPhone：mp4 / m4a 系になることがある

端末やブラウザによって形式は変わります。

### WAVで保存

作曲アプリやDAWに入れやすいよう、読み込んだ音源をWAVとして保存します。

用途：

- 作曲アプリに読み込む
- DAWに読み込む
- 後から編集する
- 他のAI作曲ツールに渡す

### 注意

WAV保存は、録音した音声を音程補正する機能ではありません。
録音した声そのものを、WAV形式で保存する機能です。

音程バーやドレミ音再生の結果を音声化して保存する機能ではなく、作曲素材として使うための「元の録音音源」の保存です。


## v011 の変更点：ドレミ音の音色改善＋ドレミ音WAV出力

### ドレミ音をキーボード風に改善

以前のドレミ音は、シンプルな電子音に近い音色でした。
v011では、以下を調整しています。

- 三角波＋倍音を重ねる
- 音の立ち上がりと減衰をキーボード風に調整
- ローパスフィルターで耳に痛い成分を抑える
- コンプレッサーで音量を整える

完全なピアノ音源ではありませんが、電子的なピー音よりも柔らかいキーボード風の音に寄せています。

### 音色切替

- キーボード風（おすすめ）
- 明るめ
- シンプル

### ドレミ音のWAV保存

解析された音程バーを、現在選択しているドレミ音色でWAV保存できます。

使い方：

1. 録音または音源読み込み
2. 解析
3. 音色を選ぶ
4. 「ドレミ音をWAV保存」

このWAVは、作曲アプリやDAWに読み込むための素材として使えます。

### 注意

これはMIDIファイルではなく、音声ファイルとしてのWAVです。
作曲アプリ側で楽器音を差し替えたい場合は、将来的にはMIDI書き出し機能を追加するのが向いています。
